# Trực Quan Hóa Toán Học với Python

Repository này tập trung vào việc trực quan hóa các khái niệm toán học bằng Python. Nó cung cấp các script và công cụ để giúp bạn hiểu và trực quan hóa các lý thuyết và bài toán khác nhau.

## Tính Năng

- Trực quan hóa các hàm và phương trình toán học phức tạp
- Đồ thị và biểu đồ tương tác
- Hướng dẫn và tutorial từng bước

## Cài Đặt

Để bắt đầu với repository này, hãy làm theo các bước sau:

1. **Clone repository:**
    ```bash
    git clone https://github.com/hungchann/learn-math.git
    cd learn-math
    ```

2. **Tạo môi trường ảo:**
    ```bash
    python3 -m venv venv
    source venv/bin/activate
    ```

3. **Cài đặt các thư viện cần thiết:**
    ```bash
    pip install -r requirements.txt
    ```

4. **Chạy các script mẫu:**
    ```bash
    python examples/example_script.py
    ```

## Cách Sử Dụng

Tham khảo thư mục `examples` để xem các script minh họa cách trực quan hóa các khái niệm toán học khác nhau. Mỗi script đều có tài liệu chi tiết để giúp bạn hiểu code và các khái niệm toán học liên quan.
